# Workshop Website (GitHub Pages)

This folder contains a standalone static website for:

**Privacy, Security, Safety and Ethics of AI in Pervasive Environments** (UbiComp/ISWC 2026 workshop draft).

## Deploy with GitHub Pages

For a repository named `ai-privacy-ubicomp.github.io`, publishing is automatic from the `main` branch root.

## Files

- `index.html` - website content
- `styles.css` - styles

## Notes

The source PDF contains placeholder/legacy information in the "Important Dates" and contact details. Update those fields before final release.
